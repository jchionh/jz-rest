Jz Rest Web Services

Available Servlets:      
Word Dict Lookup    --  Lookup service for valid English words of length 3 to 5 letters.
Example Usage: wdict/cat
Returns: 0 (not a valid word), 1 (valid word)
 
Word Dict Lookup (JSON format)    --  Same service as Word Dict Lookup, except the return result is in JSON format.
Example Usage: wdictJSON/cat
Returns: { "r":"0", "t":"11"} -- r for result, t for query time in milliseconds.
 
Jz_rest    --  a Simple Hello World test service.
